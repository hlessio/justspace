# The rules (all of them)

1. Weight → proportional area
2. Click → boost weight, siblings compress
3. Cursor → gaussian influence field
4. Row layout fixed from initial weights
5. Font scales with min(width, height)
6. Children disappear below threshold
7. Deeper clicks amplify ancestors
8. Descendants reset when parent loses focus

That's it. Eight rules. The entire UI emerges from these.
