# What we removed (and it got better every time)

- Removed discrete checkpoints (1→2→3) → made weight continuous
- Removed hover as binary state → made it a gaussian field
- Removed Back button → clicking elsewhere IS going back
- Removed Escape handler → same as clicking background
- Removed sort-by-weight in treemap → nodes stay in place
- Removed separate editing mode → editing emerges from weight
- Removed search results panel → search perturbs the weight field

Every subtraction made the system better.
