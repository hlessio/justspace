# File system integration was almost free

Folders = directories. Notes = .md files.
The tree structure IS the file system.

Open a folder → the engine reads it as a tree → instant spatial UI.

Title from # heading or filename. Body = markdown content.
YAML frontmatter for metadata (optional).
Save on edit. Create/delete = create/delete files.

Works with Obsidian vaults. Or any folder of .md files.hellooo
